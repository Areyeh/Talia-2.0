How to Fix Your Currency Dropdown

This example demonstrates a working currency dropdown menu with all options (ILS, USD, EUR, GBP) properly displayed.

Step 1: Update Your CSS

Make sure your dropdown CSS has these key properties:

.currency-selector {
    position: relative;
    display: inline-block;
}

.dropdown-menu {
    position: absolute;
    top: 100%;
    left: 0;
    display: none;
    z-index: 1000;
}

.dropdown-menu.show {
    display: block;
}

/* Remove hover behavior */
.dropdown-toggle:hover .dropdown-menu {
    display: none;
}

Step 2: Update Your JavaScript

Ensure your JavaScript properly toggles the dropdown:

document.addEventListener('DOMContentLoaded', function() {
    const currencyToggle = document.querySelector('.currency-selector .dropdown-toggle');
    const currencyMenu = document.querySelector('.currency-selector .dropdown-menu');
    
    if (currencyToggle && currencyMenu) {
        // Toggle dropdown on click
        currencyToggle.addEventListener('click', function(e) {
            e.preventDefault();
            e.stopPropagation();
            currencyMenu.classList.toggle('show');
        });
        
        // Close dropdown when clicking outside
        document.addEventListener('click', function(e) {
            if (!e.target.closest('.currency-selector')) {
                currencyMenu.classList.remove('show');
            }
        });
    }
});
